Decentralized Storage Makes the Web Robust and Resilient

Consider addressing information by what it is, not where it is. To do this we need to use a different kind of web link. Instead of using links that point to locations, we need links that uniquely describe the content itself - like a fingerprint. This content-addressed approach separates “what” from “where”, so data can flow through the network, so it can be stored and served from anywhere by anyone.
